# tic-tac-toe-app
Tic Tac Toe App

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. It allows two players to play the classic 3×3 grid game in the browser.

Features
Two player game (X and O)
Win detection
Draw detection
Reset button to restart the game
How to Run
Download or clone the project.
Open the index.html file in your browser.
Start playing Tic Tac Toe.
Technologies Used
HTML
CSS
JavaScript
